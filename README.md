# MuleSoft API Training Projects

This repository contains **learning and practice projects built using MuleSoft (Mule ESB)** as part of structured training.  
It demonstrates core concepts of **API-led connectivity**, **REST API development**, and **system/process/experience layers**.

This is an **early-career learning repository (2019)** and is preserved to showcase foundational understanding of integration and API design principles.

---

## Repository Structure


Each folder represents a separate training module or workshop exercise.

---

## What This Repository Covers

- MuleSoft fundamentals
- REST API development using Mule
- API-led connectivity concepts
- Request/response flows
- Basic transformations and routing
- Training-based API implementations

---

## Technologies Used

- MuleSoft (Mule ESB)
- Anypoint Studio
- REST APIs
- HTTP connectors
- DataWeave (basic usage)

---

## Purpose of This Repository

This repository exists to demonstrate:
- Early exposure to enterprise integration concepts
- Understanding of API design and flow-based development
- Hands-on experience with MuleSoft during training
- Learning progression over time
